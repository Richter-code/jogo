# Jogo de RPG 🗡️

Um jogo de RPG simples em texto desenvolvido em Python com sistema de combate, inventário, progressão de personagem e exploração.

## Como Jogar

### Requisitos
- Python 3.6 ou superior

### Executando o Jogo
```bash
python3 rpg_game.py
```

### Como Jogar

1. **Criação de Personagem**: Digite o nome do seu aventureiro
2. **Menu Principal**: 
   - **Explorar área**: Procure por inimigos, tesouros e aventuras
   - **Ver status**: Veja suas estatísticas atuais (vida, ataque, defesa, nível, experiência)
   - **Ver inventário**: Gerencie seus itens e ouro
   - **Descansar**: Recupere toda sua vida (custa 20 moedas de ouro)
   - **Sair**: Encerre o jogo

### Sistema de Combate

Durante um encontro com inimigos, você pode:
- **Atacar**: Cause dano baseado no seu ataque
- **Usar item**: Use poções de vida ou mana do seu inventário
- **Fugir**: Tente escapar do combate (70% de chance)

### Progressão

- **Experiência**: Ganhe EXP derrotando inimigos
- **Level Up**: Aumente automaticamente vida, ataque e defesa
- **Ouro**: Colete moedas derrotando inimigos ou explorando
- **Itens**: Use poções para se curar ou aumentar temporariamente o ataque

### Inimigos

Você pode encontrar vários tipos de inimigos:
- **Goblin**: Inimigo básico, ideal para iniciantes
- **Orc**: Mais forte, boa recompensa
- **Esqueleto**: Inimigo mágico com defesa moderada
- **Lobo**: Rápido mas com pouca defesa
- **Bandido**: Inimigo humano com boa defesa
- **Dragão Jovem**: Chefe poderoso com grande recompensa

## Executando Testes

Para verificar se o jogo está funcionando corretamente:
```bash
python3 test_rpg.py
```

## Características do Jogo

- ⚔️ Sistema de combate por turnos
- 📈 Sistema de progressão com levels
- 🎒 Sistema de inventário e itens
- 💰 Sistema de economia com ouro
- 🗺️ Exploração com encontros aleatórios
- 🛡️ Sistema de defesa e dano
- 🏠 Sistema de descanso para recuperação

## Dicas para Jogar

1. **Gerencie sua vida**: Use poções quando necessário
2. **Economize ouro**: Descanse quando estiver com pouca vida
3. **Explore bastante**: Quanto mais você explora, mais experiência ganha
4. **Use itens estrategicamente**: Poções de mana aumentam temporariamente seu ataque
5. **Fuja quando necessário**: Não há problema em fugir de combates difíceis

Divirta-se explorando este mundo de aventuras! 🎮
